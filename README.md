✅ Renan Códigos

Gerador moderno de QR Code e Código de Barras direto no navegador.

🔗 Acesse o projeto:

https://renancodigos.netlify.app/

✨ Funcionalidades
📱 Gerar QR Code a partir de qualquer texto ou link
🏷️ Gerar código de barras (formato CODE128)
📥 Download da imagem gerada
🧹 Limpar campos facilmente
💻 Interface moderna e responsiva (funciona no celular)

🛠️ Tecnologias utilizadas
HTML5
CSS
JavaScript
